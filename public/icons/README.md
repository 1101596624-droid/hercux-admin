## 开发模式

**好消息**: 开发模式不需要这些图标文件！

你可以直接运行：
```bash
npm run dev
```

应用会正常启动，只是窗口会使用 Electron 的默认图标。

## 打包模式

如果要打包应用（`npm run dist`），需要准备完整的图标文件。

请参考项目根目录的 `ICON-GUIDE.md` 了解如何准备图标。

## 重新生成图标

如果需要修改图标，编辑 `icon.svg` 文件，然后运行：

```bash
npm run generate:icons
```

或者直接运行脚本：

```bash
node scripts/generate-icons.js
```

## 图标设计

当前图标采用专业的黑灰色配色方案：
- **背景**: 深灰色渐变 (#1e293b → #334155)
- **字母 "H"**: 浅灰色渐变 (#f1f5f9 → #cbd5e1)
- **装饰线条**: 中灰色 (#64748b)
- **底部装饰点**: 浅灰色 (#94a3b8)

这个配色方案体现了 HERCU 的专业性和现代感，适合桌面应用使用。

如需自定义图标，请编辑 `icon.svg` 文件。
